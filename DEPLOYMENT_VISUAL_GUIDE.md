## 🐛 حل المشاكل الشائعة

### المشكلة 1: الصفحة لا تظهر (404)
```
❌ المشكلة: "404 Not Found"

✅ الحل:
1. تأكد من وجود index.html في public_html/
2. تأكد من أن الملفات ليست داخل مجلد dist/
3. تحقق من الصلاحيات (644 للملفات)
```

### المشكلة 2: الصفحة بدون تنسيق
```
❌ المشكلة: الصفحة تظهر بدون ألوان أو تصميم

✅ الحل:
1. تأكد من رفع مجلد assets/ كاملاً
2. افتح Developer Tools (F12)
3. تحقق من وجود أخطاء في Console
4. امسح الكاش (Ctrl+Shift+R)
```

### المشكلة 3: الصور لا تظهر
```
❌ المشكلة: الصور لا تظهر

✅ الحل:
1. تأكد من رفع مجلد images/ كاملاً
2. تحقق من صلاحيات المجلد (755)
3. امسح الكاش
```

### المشكلة 4: HTTPS لا يعمل
```
❌ المشكلة: الموقع يفتح على HTTP بدلاً من HTTPS

✅ الحل:
1. ثبّت شهادة SSL من cPanel
2. تأكد من وجود ملف .htaccess
3. تحقق من قاعدة Force HTTPS في .htaccess
4. امسح الكاش
```
