# بيانات المناطق والمدن والأحياء (السعودية)

الملفات `*_lite.json` مأخوذة من المشروع:

**Saudi-Arabia-Regions-Cities-and-Districts** — [homaily على GitHub](https://github.com/homaily/Saudi-Arabia-Regions-Cities-and-Districts)

- المصدر الأصلي: بيانات العنوان الوطني (maps.address.gov.sa) كما هو موثّق في المستودع.
- الترخيص: **GPL-2.0** — عند التوزيع يجب الالتزام بشروط GPL.

نسخة **lite** (بدون حدود جغرافية مفصّلة) لتقليل الحجم مع الإبقاء على أسماء المناطق والمدن والأحياء بالعربية.
