## ⚡ خطوات التحديث السريعة

### 1️⃣ **بناء المشروع**

```bash
# الانتقال إلى مجلد المشروع
cd halaqmap

# بناء المشروع
npm run build
```

### 2️⃣ **النسخ الاحتياطي**

**⚠️ مهم جداً: احتفظ بنسخة احتياطية قبل التحديث!**

#### في cPanel:
```
1. افتح File Manager
2. حدد مجلد public_html/
3. اضغط "Compress" أو "ضغط"
4. اختر ZIP
5. حمّل الملف المضغوط على جهازك
6. احتفظ به في مكان آمن
```

### 3️⃣ **حذف الملفات القديمة**

```
⚠️ لا تحذف ملف .htaccess!

في public_html/:
✅ احذف: assets/
✅ احذف: images/
✅ احذف: index.html
✅ احذف: favicon.ico
✅ احذف: robots.txt
❌ لا تحذف: .htaccess
```

### 4️⃣ **رفع الملفات الجديدة**

```
1. افتح File Manager
2. انتقل إلى public_html/
3. ارفع جميع الملفات من dist/ (ما عدا .htaccess إذا كان موجوداً)
4. انتظر حتى يكتمل الرفع
```

### 5️⃣ **مسح الكاش**

#### في المتصفح:
```
Chrome/Edge: Ctrl + Shift + R
Firefox: Ctrl + Shift + R
Safari: Cmd + Shift + R
```

#### في Cloudflare (إذا كنت تستخدمه):
```
1. سجّل الدخول إلى Cloudflare
2. اختر halaqmap.com
3. اذهب إلى Caching
4. اضغط "Purge Everything"
```

### 6️⃣ **اختبار التحديث**

```
✅ افتح: https://halaqmap.com
✅ تحقق من الصفحة الرئيسية
✅ اختبر جميع الصفحات
✅ تحقق من Console (F12) للأخطاء
✅ اختبر على الموبايل
```

## 🔧 سيناريوهات التحديث الشائعة

### 📝 تحديث النصوص فقط

```
إذا قمت بتعديل نصوص فقط (بدون تغيير في التصميم):

1. npm run build
2. ارفع index.html فقط
3. امسح الكاش
4. اختبر
```

### 🎨 تحديث التصميم (CSS)

```
إذا قمت بتعديل التصميم:

1. npm run build
2. احذف مجلد assets/ القديم
3. ارفع مجلد assets/ الجديد
4. ارفع index.html
5. امسح الكاش
6. اختبر
```

### 🖼️ إضافة صور جديدة

```
إذا أضفت صور جديدة:

1. npm run build
2. ارفع مجلد images/ الجديد
3. ارفع index.html
4. ارفع assets/ (إذا تغير)
5. اختبر
```

### ⚙️ تحديث الوظائف (JavaScript)

```
إذا قمت بتعديل الوظائف:

1. npm run build
2. احذف مجلد assets/ القديم
3. ارفع مجلد assets/ الجديد
4. ارفع index.html
5. امسح الكاش
6. اختبر بعناية
```

### 🆕 إضافة صفحة جديدة

```
إذا أضفت صفحة جديدة:

1. npm run build
2. ارفع جميع الملفات من dist/
3. تأكد من تحديث .htaccess إذا لزم الأمر
4. امسح الكاش
5. اختبر الصفحة الجديدة
```

## 🐛 حل مشاكل التحديث

### المشكلة 1: التحديث لا يظهر

```
❌ المشكلة: الموقع يظهر النسخة القديمة

✅ الحل:
1. امسح الكاش بقوة (Ctrl+Shift+R)
2. جرّب في وضع التصفح الخفي
3. امسح كاش Cloudflare (إذا كنت تستخدمه)
4. انتظر 5-10 دقائق
5. تحقق من أن الملفات الجديدة مرفوعة فعلاً
```

### المشكلة 2: أخطاء بعد التحديث

```
❌ المشكلة: الموقع لا يعمل بعد التحديث

✅ الحل:
1. افتح Console (F12)
2. اقرأ رسائل الأخطاء
3. تحقق من أن جميع الملفات مرفوعة
4. تحقق من أن assets/ موجود
5. إذا فشل كل شيء، استعد النسخة الاحتياطية
```

### المشكلة 3: الصور لا تظهر

```
❌ المشكلة: الصور القديمة تظهر أو لا تظهر

✅ الحل:
1. تحقق من رفع مجلد images/
2. تحقق من صلاحيات المجلد (755)
3. امسح الكاش
4. تحقق من مسارات الصور في الكود
```
