# 🚀 رفع موقع حلاق ماب - دليل سريع

## 📦 الملفات الجاهزة للرفع

جميع الملفات الجاهزة موجودة في مجلد: **`dist/`**

---

## ⚡ خطوات سريعة (5 دقائق)

### 1️⃣ **تحميل الملفات**
- اضغط على زر **Download** (↓) في أعلى الصفحة
- أو استخدم الملفات من مجلد `dist/`

### 2️⃣ **رفع الملفات على الاستضافة**

#### **باستخدام cPanel:**
```
1. سجّل الدخول إلى cPanel
2. افتح File Manager
3. انتقل إلى public_html/
4. احذف الملفات القديمة
5. ارفع جميع الملفات من داخل مجلد dist/
```

#### **باستخدام FTP (FileZilla):**
```
1. اتصل بالسيرفر عبر FTP
2. افتح مجلد public_html/
3. اسحب جميع الملفات من dist/ إلى public_html/
```

### 3️⃣ **تفعيل SSL**
```
1. في cPanel > SSL/TLS Status
2. اختر halaqmap.com
3. اضغط Install (Let's Encrypt مجاني)
```

### 4️⃣ **اختبار الموقع**
```
افتح: https://halaqmap.com
```

---

## 📋 البنية الصحيحة للملفات

```
public_html/
├── index.html          ← ملف HTML الرئيسي
├── .htaccess          ← ملف الإعدادات (مهم!)
├── assets/            ← ملفات CSS و JS
│   ├── index-xxx.css
│   ├── index-xxx.js
│   └── ...
└── images/            ← الصور
    └── ...
```

**⚠️ مهم:** الملفات يجب أن تكون **داخل** `public_html/` مباشرة، وليس داخل `public_html/dist/`

---

## 🔧 ملف .htaccess (مهم جداً!)

ملف `.htaccess` موجود بالفعل في مجلد `dist/`

**إذا لم يُرفع تلقائياً:**
1. أنشئ ملف جديد اسمه `.htaccess` في `public_html/`
2. انسخ المحتوى من `dist/.htaccess`
3. احفظ الملف

## 🐛 حل المشاكل

### **الصفحة لا تظهر:**
- تأكد من وجود `index.html` في `public_html/`
- تأكد من أن الملفات ليست داخل مجلد فرعي

### **CSS لا يعمل:**
- تأكد من رفع مجلد `assets/` كاملاً
- امسح الكاش (Ctrl+Shift+R)

### **HTTPS لا يعمل:**
- ثبّت شهادة SSL من cPanel
- تأكد من وجود ملف `.htaccess`
