# 🎯 دليل مصور: كيف ترفع المجلدات بالطريقة الصحيحة

---

## 📋 الإجابة المباشرة

### **السؤال: هل أرفع المجلدات أم المحتويات؟**

**الجواب:**
- **على Vercel/Netlify:** ارفع المجلد الرئيسي كاملاً ✅
- **على cPanel:** ارفع **محتويات** `dist/` فقط (بدون المجلد نفسه) ✅

---

## 🎬 السيناريو 1: الرفع على cPanel (الأهم!)

### **المشكلة:**

```
لديك مجلد dist/ يحتوي على:
dist/
├── index.html
├── .htaccess
├── assets/
└── images/
```

### **السؤال:**
هل أرفع مجلد `dist/` كاملاً؟ أم محتوياته فقط؟

### **الجواب:**
❌ **لا ترفع المجلد!**
✅ **ارفع المحتويات فقط!**

## 📦 الطريقة الصحيحة خطوة بخطوة

### **الخطوة 1: افتح مجلد dist/**

```
في الكمبيوتر:
halaqmap/
└── dist/              ← افتح هذا المجلد
    ├── index.html
    ├── .htaccess
    ├── assets/
    └── images/
```

### **الخطوة 2: حدد جميع المحتويات (وليس المجلد نفسه)**

```
داخل مجلد dist/:
✅ حدد: index.html
✅ حدد: .htaccess
✅ حدد: assets/ (المجلد كاملاً)
✅ حدد: images/ (المجلد كاملاً)

⚠️ لا تحدد مجلد dist/ نفسه!
```

**كيف تحدد الكل؟**
- **Windows:** `Ctrl + A`
- **Mac:** `Cmd + A`
- **أو:** اسحب الماوس على جميع الملفات

### **الخطوة 3: اضغط الملفات**

#### **على Windows:**
```
1. بعد تحديد جميع الملفات داخل dist/
2. اضغط بزر الماوس الأيمن
3. اختر "Send to" → "Compressed (zipped) folder"
4. سمّه: halaqmap-files.zip
```

#### **على Mac:**
```
1. بعد تحديد جميع الملفات داخل dist/
2. اضغط بزر الماوس الأيمن
3. اختر "Compress X items"
4. سمّه: halaqmap-files.zip
```

#### **على Linux:**
```bash
cd halaqmap/dist/
zip -r ../halaqmap-files.zip *
```

### **الخطوة 4: ارفع على cPanel**

```
1. سجّل الدخول إلى cPanel
2. افتح "File Manager"
3. انتقل إلى public_html/
4. احذف الملفات القديمة (احتفظ بنسخة احتياطية!)
5. اضغط "Upload"
6. ارفع halaqmap-files.zip
7. انتظر حتى ينتهي الرفع
```

### **الخطوة 5: فك الضغط**

```
1. في File Manager، انتقل إلى public_html/
2. اضغط بزر الماوس الأيمن على halaqmap-files.zip
3. اختر "Extract"
4. تأكد من أن المسار هو: /public_html/
5. اضغط "Extract Files"
6. احذف halaqmap-files.zip
```

### **الخطوة 6: التحقق**

```
✅ الصحيح:
public_html/
├── index.html          ← موجود هنا مباشرة
├── .htaccess          ← موجود هنا مباشرة
├── assets/            ← مجلد هنا مباشرة
└── images/            ← مجلد هنا مباشرة

❌ خطأ:
public_html/
└── dist/              ← مجلد إضافي! خطأ!
    ├── index.html
    └── ...
```

## 🔧 إصلاح الخطأ إذا رفعت المجلد بالخطأ

### **إذا رفعت مجلد dist/ بالخطأ:**

```
الوضع الحالي (خطأ):
public_html/
└── dist/
    ├── index.html
    └── ...

الحل:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. في File Manager، انتقل إلى: public_html/dist/
2. حدد جميع الملفات داخل dist/ (Ctrl+A)
3. اضغط "Move" في الأعلى
4. في نافذة Move:
   - اختر المسار: /public_html/
   - اضغط "Move Files"
5. ارجع إلى public_html/
6. احذف مجلد dist/ الفارغ

النتيجة (صحيح):
public_html/
├── index.html
├── .htaccess
├── assets/
└── images/

✅ الآن الموقع يعمل!
```

## 🎯 السيناريو 2: الرفع على Vercel (أسهل!)

### **على Vercel، الأمر مختلف:**

```
✅ ارفع المجلد الرئيسي كاملاً:

halaqmap/              ← ارفع هذا المجلد كاملاً على GitHub
├── src/               ← سيتم رفعه
├── public/            ← سيتم رفعه
├── package.json       ← سيتم رفعه
└── ...

❌ لا ترفع:
├── node_modules/      ← لا ترفعه!
└── dist/              ← لا ترفعه!

لماذا؟
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Vercel سيقوم بـ:
1. تثبيت node_modules/ تلقائياً
2. بناء المشروع (npm run build)
3. إنشاء dist/ تلقائياً
4. نشر الموقع

✅ لا تحتاج للقلق بشأن المجلدات!
```

## 🆘 الأسئلة الشائعة

### **س1: هل أرفع مجلد assets/ أم محتوياته؟**

```
على cPanel:
✅ ارفع مجلد assets/ كاملاً (مع محتوياته)

النتيجة:
public_html/
└── assets/          ← المجلد كاملاً
    ├── index-xxx.css
    └── index-xxx.js
```

### **س2: هل أرفع مجلد dist/ أم محتوياته؟**

```
على cPanel:
❌ لا ترفع مجلد dist/
✅ ارفع محتويات dist/ فقط

النتيجة:
public_html/
├── index.html       ← من داخل dist/
├── assets/          ← من داخل dist/

```

### **س3: لماذا الموقع لا يعمل بعد الرفع؟**

```
السبب الأكثر شيوعاً:
❌ رفعت مجلد dist/ كاملاً

الحل:
1. انقل محتويات dist/ إلى public_html/
2. احذف مجلد dist/
```
