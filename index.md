### 🎨 أيقونة التبويب (Favicon) — شعار المنصة في المتصفح

```
⏱️ الوقت: دقيقتان
📊 المستوى: الجميع
📝 الوصف: إظهار شعار حلاق ماب في تبويب المتصفح عند الرفع
✅ الأفضل لـ: من يرفع الموقع ويريد أيقونة العلامة بدل الأيقونة الافتراضية
📌 يشمل:
   • الملفات المطلوبة في المشروع
   • كود HTML الذي يجب أن يكون داخل index.html
   • ماذا ترفع بعد npm run build
```

**الملفات في المشروع (قبل/بعد البناء):**

| المسار | الدور |
|--------|--------|
| `public/favicon.svg` | ملف الأيقونة (متجه، ألوان تيل + ذهب) |
| `index.html` | يحتوي روابط الأيقونة داخل `<head>` |

**انسخ هذا الكود داخل `<head>` في `index.html` (أو تأكد أنه موجود كما هو):**

```html
<meta name="theme-color" content="#0d9488" />
<link rel="icon" type="image/svg+xml" href="/favicon.svg" />
<link rel="alternate icon" href="/favicon.svg" />
```

**عند الرفع على الاستضافة:**

1. نفّذ `npm run build`.
2. ارفع محتويات مجلد `dist/` إلى جذر الموقع.
3. تأكد أن **`favicon.svg`** يظهر **في نفس مستوى** `index.html` الناتج عن البناء (Vite ينسخ محتويات `public/` إلى جذر `dist/` تلقائياً).
4. إذا رفعت يدوياً بدون مجلد `public`، انسخ `favicon.svg` من `public/` إلى نفس مجلد `index.html` على السيرفر.

**لاستبدال الشعار بصورة خاصة بك:** استبدل الملف `public/favicon.svg` بشعارك (يفضّل SVG أو PNG)، أو غيّر `href="/favicon.svg"` إلى اسم ملفك (مثل `/favicon.png`) وأضف `type="image/png"` إن لزم.
